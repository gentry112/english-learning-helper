# 英语学习助手（六级 + 考研 整合版）

面向 CET-6 和考研英语一的 AI 学习工具。单文件网页，浏览器打开即用。

> **个人学习项目**：由个人独立构思，借助 AI 编程工具实现。所有功能设计、交互流程、Prompt 工程均基于自身备考需求打磨。

---

## 考试类型

顶部渐变栏一键切换：

- **📚 六级**（绿色）— 词汇查询 · 中译英 · 大作文
- **📖 考研**（蓝色）— 词汇查询 · 英译中 · 小作文 · 大作文

切换后文件夹数据、模式 Tab、AI Prompt、历史记录**完全隔离**。

---

## 功能总览

### 📖 词汇查询（六级 / 考研共用）

输入英文单词、短语或短句 → AI 返回五分区卡片：

| 分区 | 内容 |
|---|---|
| 📝 拼写检查 | 正确通过 / 错误则纠正建议，橙色高亮 |
| 📖 释义 | 短语/句子整体释义 + 逐词拆解（原形、变形、词性×释义、派生词） |
| 📊 频率 | 按考试类型标注 🔴高频 / 🟡中频 / 🟢低频，附真题出现说明 |
| 📝 学习记录 | 首次查询 / 累计查询 X 次 |
| 📚 拓展 | 常考搭配、真题例句、近义词辨析、写作高级替换 |

**长文本保护**：输入超过 15 个英文单词时自动提示切换到翻译精读模式。

### 🈂️ 翻译训练（六级中译英 · 考研英译中）

翻译模式内分两个子模式，胶囊按钮切换：

**📝 翻译题目解析**（默认）：两步交互式学习
1. 输入句子 → AI 解析句子结构 + 翻译要点
2. 选择"批改我的翻译"或"查看示例译文"
3. 批改：用户提交译文 → AI 评分 + 分维度评价 + 逐句修改建议
4. 示例：AI 展示简单拿分版（带得分档位）+ 进阶高分版（带亮点和知识点）

**📖 阅读段落精读**：一步快速分析
- 粘贴长段落 → AI 一次性返回：
  - 📖 全文翻译
  - 📝 逐句对照（原文 + 译文 + 句式结构说明）
  - 🔑 核心词汇表（5-10 个重难词）
  - 🔬 长难句精析（2-3 个最复杂句子逐层拆解）

### 📝 作文辅助

**六级大作文 / 考研小作文**（议论文 / 应用文）：

1. 输入题目 → AI 解析题目（类型、要点、结构建议、避坑指南）
2. 选择"批改我的作文"或"直接生成模板"
3. 批改：用户提交作文 → AI 评分 + 五维度评价 + 逐段问题修改建议 + 修改后范文
4. 模板：可选输入关键词 → AI 生成 **📗 简单版**（基础词汇 + `[...]` 填空）和 **📘 进阶版**（高级句式 + 地道表达）写作模板

**考研大作文**（图画/图表作文）：

1. 进入模式 → 直接显示结构化描述表单
2. 选择类型：🎨 图画 / 📊 图表
3. **🖼️ 拍照识图**：调用通义千问 VL 分析图片内容，自动填入表单各字段
4. 或手动填写表单（人物/主体、动作/场景、寓意 / 图表类型、数据、趋势等）
5. 确认 → AI 解析题目 → 后续流程同上

### 📊 历史记录

点击顶部 📊 进入，按考试类型动态生成 Tab：

- 词汇 Tab：按综合权重排序（查询次数×0.4 + 频率×0.6）
- 翻译 / 作文 Tab：按时间倒序
- 点击记录卡片可恢复查看完整内容（重新调用 AI 结果缓存，不消耗 API）
- 支持单条删除、清空全部，底部 📥 导出全部数据为 Markdown

### 📂 文件夹管理

点击顶部 📂 管理文件夹，按真题篇目分类存放学习记录。支持新建、切换、删除（至少保留一个）。

### 📷 拍照识别文字

底部 📷 按钮（通义千问 VL OCR）：拍英语文章 → 识别文字 → 可编辑修正 → 填入输入框。

### ⚙️ 设置

- DeepSeek API Key（主 AI，需自行注册 [platform.deepseek.com](https://platform.deepseek.com/)）
- 通义千问视觉识别 Key（拍照 OCR + 考研大作文识图，[dashscope.aliyun.com](https://dashscope.aliyun.com/)）
- 测试连接、导出全量数据 Markdown、清除数据
- 首次打开未设置 Key 时自动弹出功能介绍弹窗

## 使用方法

1. 浏览器打开 `index.html`（推荐手机端，微信可直接打开）
2. 首次使用弹出欢迎弹窗 → 点击设置 API Key
3. 切换考试类型、文件夹，开始学习

---

## 技术栈

| 层级 | 技术 |
|---|---|
| 框架 | Vue 3（CDN） |
| 样式 | Tailwind CSS（CDN Play） |
| AI 主引擎 | DeepSeek API（`deepseek-chat`，前端直调） |
| AI 视觉 | 通义千问 VL API（`qwen-vl-plus`，OCR + 识图） |
| 存储 | localStorage（数据本地化，不上传服务器） |
| 部署 | 单文件 HTML，无后端，无构建工具 |

---

## 数据存储

| localStorage Key | 内容 |
|---|---|
| `cet6-api-key` | DeepSeek API Key（六级/考研共用） |
| `ocr-api-key` | 通义千问视觉识别 API Key |
| `exam-type` | 当前考试类型 `'cet6'` / `'postgrad'` |
| `cet6-data` | 六级全部数据（文件夹 + 历史记录） |
| `postgrad-data` | 考研全部数据（文件夹 + 历史记录） |

旧版 `cet6-folders` 首次打开时自动迁移。

## 错误处理

| 场景 | 提示 |
|---|---|
| API Key 缺失 | 自动弹出欢迎弹窗 / 设置页 |
| API Key 无效 (401) | 提示前往设置更新 |
| 请求频繁 (429) | 提示稍后重试 |
| 网络错误 | 提示检查网络 |
| 超时 (30s) | 提示超时重试 |
| AI 返回格式异常 | 提示重试 |
| 词汇模式输入非英文 | 提示仅支持英文 |
| 词汇模式输入过长 | 引导切换到段落精读 |
| 空输入 | 输入框抖动 + 红色提示 |

---

## 限制

- 纯前端，无后端服务器
- 数据存浏览器 localStorage，清除数据会丢失
- 依赖第三方 API（DeepSeek + 通义千问），需用户自行提供 Key
- 无账号系统、无云端同步、无离线模式
