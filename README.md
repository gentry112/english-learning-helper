# 英语学习助手（六级 + 考研 整合版）

面向 CET-6 和考研英语一备考学生的移动端网页学习工具。一个链接即可在六级和考研两种考试类型间自由切换。

## 考试类型切换

顶部蓝色切换栏选择当前考试类型：

- **📚 六级**：词汇查询、中译英、大作文（3 模式）
- **📖 考研**：词汇查询、英译中、小作文、大作文（4 模式）

切换后文件夹数据、模式 Tab、AI Prompt、历史记录全部独立隔离。

## 功能

### 📖 词汇查询（六级/考研共用）
- 拼写检查与纠错
- 完整释义（原形、词性、派生词）
- 考试频率标注（高频🔴 / 中频🟡 / 低频🟢）
- 常考搭配、真题例句、近义词辨析、写作高级表达

### 六级专属
- **🈂️ 中译英**：句子结构解析 + 简单拿分版 + 进阶高分版
- **📝 大作文**：议论文题目解析 + 双版本写作模板

### 考研专属
- **🈂️ 英译中**：长难句结构解析 + 通顺版译文 + 精准版译文
- **✉️ 小作文**：应用文（书信/通知/备忘录）题目解析 + 双版本模板
- **📝 大作文**：图画/图表作文题目解析 + 双版本模板

## 使用方法

1. 用浏览器打开 `index.html`
2. 点击顶部蓝色栏切换六级/考研
3. 点击右上角 ⚙️ 设置 DeepSeek API Key
4. 开始学习

## 技术栈

- Vue 3（CDN）
- Tailwind CSS（CDN Play）
- DeepSeek API（前端直调）
- localStorage 数据持久化
- 单文件 HTML，无构建工具，无后端

## 数据存储

| Key | 内容 |
|-----|------|
| `cet6-api-key` | DeepSeek API Key（共用） |
| `exam-type` | 当前考试类型 'cet6' / 'postgrad' |
| `cet6-data` | 六级文件夹与历史数据 |
| `postgrad-data` | 考研文件夹与历史数据 |

旧版 `cet6-folders` 会在首次打开时自动迁移。
