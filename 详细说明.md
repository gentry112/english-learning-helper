# 英语学习助手 — 详细技术说明

## 一、项目概述

单文件 HTML 网页应用（~4400 行），面向 CET-6 和考研英语一备考学生。浏览器直接打开即用，无需安装。

- **文件名**：index.html
- **访问方式**：浏览器打开（推荐移动端，微信可直接打开）
- **版本**：整合版 v3.0

---

## 二、考试类型切换

### 2.1 切换栏

页面顶部渐变栏：
- 📚 六级：绿色渐变（`from-green-500 to-emerald-500`）
- 📖 考研：蓝色渐变（`from-blue-500 to-indigo-500`）

切换后完整重置：页面标题、模式 Tab、localStorage 数据、聊天消息、历史 Tab。

### 2.2 数据隔离

| localStorage Key | 内容 |
|---|---|
| `cet6-api-key` | DeepSeek API Key（共用） |
| `ocr-api-key` | 通义千问 VL API Key（共用） |
| `exam-type` | 当前考试类型 |
| `cet6-data` | 六级全部数据 |
| `postgrad-data` | 考研全部数据 |

旧版 `cet6-folders` 首次打开自动迁移到新 Key 并删除。

### 2.3 欢迎弹窗

首次打开未设置 API Key 时，自动弹出功能介绍弹窗（5 功能点 + 技术栈说明），可选择"去设置"或"稍后再说"。

## 三、考试类型配置（EXAM_CONFIG）

核心架构：**配置驱动**。两种考试类型的行为差异全部定义在 `EXAM_CONFIG` 对象中，模板和逻辑通过 `computed(() => EXAM_CONFIG[state.examType])` 动态读取。

### 3.1 六级（cet6）

| 模式 Key | 功能 | 消息类型 |
|---|---|---|
| `vocab` | 📖 词汇查询 | result |
| `translate` | 🈂️ 中译英 | translate-analysis → translate-result / translate-grade-result |
| `essay` | 📝 大作文 | essay-analysis → essay-result / essay-grade-result |

### 3.2 考研（postgrad）

| 模式 Key | 功能 | 消息类型 |
|---|---|---|
| `vocab` | 📖 词汇查询 | result |
| `en-to-cn` | 🈂️ 英译中 | translate-analysis → en-to-cn-result / translate-grade-result |
| `short-essay` | ✉️ 小作文 | short-essay-analysis → short-essay-result / essay-grade-result |
| `long-essay` | 📝 大作文 | long-essay-form → long-essay-analysis → long-essay-result / essay-grade-result |

### 3.3 配置字段说明

每个考试类型的配置包含：`label`、`title`、`storageKey`、`frequencyLabel`、`modes`（含 `isEssayMode`、`isTranslationMode`、`translationField`、`inputValidation`）、`historyTabs`、`historyTabLabels`、`welcome`、`placeholder`、`emptyHint`、`keywordGuide`。

## 四、核心功能详解

### 4.1 词汇查询

**流程**：输入英文 → `sendVocabQuery()` → `buildSystemPrompt()` → `callDeepSeek()` → JSON.parse → 渲染 result 气泡

**返回结构**：
```json
{
  "spellCheck": { "isCorrect": true/false, "correction": "", "note": "" },
  "definition": {
    "overall": "整体释义（短语/句子时）",
    "words": [{ "word": "原词", "rootForm": "原形", "inflection": "变形说明",
                "meanings": [{ "pos": "词性", "meaning": "释义" }],
                "derivatives": [{ "word": "派生词", "pos": "词性", "meaning": "释义" }] }]
  },
  "cetFrequency": { "level": "高频/中频/低频", "detail": "" },
  "extension": {
    "collocations": [{ "en": "", "zh": "" }],
    "exampleSentence": { "en": "", "zh": "" },
    "synonyms": [{ "word": "", "pos": "", "difference": "" }],
    "advancedAlternatives": [{ "word": "", "pos": "", "meaning": "" }]
  }
}
```

**长文本保护**：超过 15 个英文单词时提示切换到翻译精读模式，避免大 JSON 导致解析失败。

**历史权重**：`权重 = 查询次数 × 0.4 + 频率分 × 0.6`（高频=3/中频=2/低频=1）

### 4.2 翻译模式

#### 子模式切换

翻译模式内部（translate / en-to-cn）增加一行胶囊按钮：
- **📝 翻译题目解析**（默认蓝色）— 两步交互
- **📖 阅读段落精读**（紫色）— 一步输出

切换时清理上下文 + 刷新欢迎消息 + Toast 提示当前模式。

#### 翻译题目解析（analysis）

**第一步**：输入句子 → `sendTranslateQuery()`/`sendEnToCnQuery()` → AI 返回解析（`analysis.segments[]`）→ 保存到 `state.translateContext` → 渲染 translate-analysis 气泡（含批改/生成按钮）

**第二步（批改）**：用户点击 → `postAnalysisAction = 'grade'` → 底部输入栏切换为作文输入 → 提交 → `submitGrading()` → `buildTranslateGradePrompt()` → 返回评分结构 → 渲染 translate-grade-result 气泡

**第二步（生成）**：用户点击 → `postAnalysisAction = 'generate'` → 直接从缓存 `translateContext.translationPairs` 渲染 translate-result 气泡

**翻译结果结构**：
```json
{
  "analysis": { "segments": [{ "original": "", "structure": "", "approach": "" }] },
  "translationPairs": [{
    "original": "原文句子",
    "basicVersions": [{ "text": "简单版", "score": "10-12分/15分制", "tips": "" }],
    "advancedVersions": [{ "text": "进阶版", "highlights": [], "knowledgePoints": [] }]
  }]
}
```

模板同时兼容新格式（`translationPairs`）和旧格式（顶层 `basicVersions`/`advancedVersions`）。

#### 阅读段落精读（intensive-reading）

一步输出：`sendTranslateQuery()`/`sendEnToCnQuery()` 检测 `translationSubMode === 'intensive-reading'` → 调用 `buildIntensiveReadingPrompt()` → 返回：

```json
{
  "fullTranslation": "全文翻译",
  "sentenceAnalysis": [{ "original": "", "translation": "", "structure": "" }],
  "keyVocabulary": [{ "word": "", "pos": "", "pronunciation": "", "meaning": "", "example": "" }],
  "difficultSentences": [{ "sentence": "", "analysis": "", "translation": "" }]
}
```

渲染 `intensive-reading-result` 气泡 — 四段式卡片。

#### 批改评分

通用 `buildTranslateGradePrompt(isEnToCn)` 函数，接收原文和用户译文，返回：
```json
{
  "overallScore": 85,
  "scoreBreakdown": [{ "dimension": "", "score": 20, "maxScore": 25, "comment": "" }],
  "issues": [{ "location": "", "sourceText": "", "original": "", "problem": "", "suggestion": "", "suggestionZh": "" }],
  "improvedVersion": { "segments": [{ "original": "", "translation": "" }] },
  "overallComment": ""
}
```

### 4.3 作文模式

#### 六级大作文 / 考研小作文

**通用两步流程**：

1. 输入题目 → `sendEssayTopicQuery()` → `buildEssayTopicPromptByMode()` → AI 返回解析 → 保存到 `state.essayContext` → 渲染 essay-analysis 气泡

2. 选择操作：
   - 批改：`postAnalysisAction = 'grade'` → 用户提交作文 → `submitGrading()` → `buildEssayGradePrompt()` → 渲染 essay-grade-result
   - 生成：`postAnalysisAction = 'generate'` → 显示关键词引导 → 用户输入关键词 → `sendEssayWithKeywords()` → 双版模板

**作文解析结构**：
```json
{
  "topicAnalysis": {
    "type": "议论文/应用文",
    "keyPoints": ["要点1"],
    "structure": "建议结构",
    "format": "格式结构（小作文）",
    "traps": ["避坑指南"]
  }
}
```

**模板结构**：
```json
{
  "basicTemplate": { "paragraphs": [{ "section": "", "en": "", "zh": "", "knowledge": "" }] },
  "advancedTemplate": { "paragraphs": [{ "section": "", "en": "", "zh": "", "knowledge": "" }] },
  "keywordsUsed": ["使用说明"]
}
```

**批改评分**（五维度）：
```json
{
  "overallScore": 80,
  "scoreBreakdown": [
    { "dimension": "内容切题/结构逻辑/语法准确性/词汇丰富度/任务完成度", "score": 0, "maxScore": 20 }
  ],
  "issues": [{ "location": "", "original": "", "problem": "", "suggestion": "", "suggestionZh": "" }],
  "improvedVersion": { "paragraphs": [{ "en": "", "zh": "" }] },
  "overallComment": ""
}
```

#### 考研大作文（long-essay）

**流程**：

1. 进入模式 → 直接显示 `long-essay-form` 气泡（结构化表单）
2. 选择类型：🎨 图画 / 📊 图表（`longEssayType`）
3. **🖼️ 拍照识图**：`openEssayCamera()` → 压缩图片 → `analyzeImage(base64, prompt)` → 千问 VL 返回结构文本 → 弹出调试窗口 `showVLResult` → 用户编辑修正 → `confirmVLResult()` → `parseVLResult()` 解析并填入表单

**VL Prompt**（`buildVLImagePrompt()`）：严格逐行格式，区分图画和图表输出不同字段

**VL 解析**（`parseVLResult()`）：
- Markdown 清理（去 `**`、`-`、`#`）
- 关键词匹配（`includes` 容错）
- 兜底提取（额外说明无结构化字段时从中提取人物/动作）

4. 手动填写或 VL 填充 → [确认] → `submitLongEssayForm()` → `buildLongEssayTopicFromForm()` 拼接描述 → `buildLongEssayTopicPrompt()` → AI 解析 → 渲染 `long-essay-analysis` 气泡

5. 后续流程同六级大作文（批改/生成模板）

**表单字段**：
- 图画：`subjects`、`action`、`meaning`、`description`
- 图表：`chartType`、`chartTitle`、`dataPoints`、`trend`、`phenomenon`、`description`
- 通用：`extraKeywords`（解析后使用）

## 五、文件夹与历史记录

### 5.1 文件夹

- 每个考试类型独立管理
- 默认"默认"文件夹
- 支持新建、切换、删除（至少保留一个）
- 切换文件夹清空聊天消息

### 5.2 历史记录视图

| 考试类型 | Tab 列表 |
|---|---|
| 六级 | 📖 词汇 / 🈂️ 翻译 / 📝 作文 |
| 考研 | 📖 词汇 / 🈂️ 英译中 / ✉️ 小作文 / 📝 大作文 |

- 词汇按综合权重降序，其余按时间倒序
- 精读记录显示紫色"精读"徽标
- 点击恢复完整内容（从 `lastResult`/`gradeRecord`/`translationPairs` 缓存重建，不调 API）
- 导出全量数据为 Markdown

## 六、技术架构

### 6.1 技术栈

| 层 | 技术 |
|---|---|
| 框架 | Vue 3（CDN `unpkg.com/vue@3`） |
| 样式 | Tailwind CSS（CDN Play） |
| 主 AI | DeepSeek API（`deepseek-chat`，`temperature: 0.3`，`response_format: json_object`，30s 超时） |
| 视觉 AI | 通义千问 VL（`qwen-vl-plus`，25s 超时） |
| 存储 | localStorage |
| 部署 | 单文件 HTML |

### 6.2 颜色系统

| 变量 | 色值 | 用途 |
|---|---|---|
| `wechat-green` | `#95EC69` | 用户气泡 + 选中态 |
| `cet-orange` | `#FF6B35` | 高频标记 + 批改按钮 |
| `bg-gray` | `#F5F5F5` | 聊天背景 |
| `text-dark` | `#333333` | 主文字 |
| `border-light` | `#E5E5E5` | 边框 |

### 6.3 CSS 动画

- **加载点**：1.4s 循环，逐点 0.2s 延迟
- **输入框抖动**：空输入 0.3s shake
- **Toast**：纵向滑入/滑出 0.3s
- **弹窗**：缩放入场 0.92→1
- **消息**：上移淡入 0.25s

### 6.4 状态结构

```
state = {
  examType, currentView, currentMode, summaryTab,
  currentFolderId, folders, messages,
  inputText, inputError, isSending,
  isOcrProcessing, ocrResultText, showOcrResult,
  essayContext, essayFormData, translationSubMode,
  translateContext, postAnalysisAction, gradingUserAnswer,
  showSettings, apiKey, apiKeyVisible, ocrApiKey, ocrApiKeyVisible,
  isTesting, showVLResult, vlResultRaw, showWelcomeModal,
  showFolderDropdown, showNewFolderDialog, newFolderName,
  toast, confirmDialog
}
```

### 6.5 Prompt 体系（16 套）

| 函数 | 方向 | 考试类型 |
|---|---|---|
| `buildSystemPrompt()` | 通用 | 词汇查询 |
| `buildTranslatePrompt()` | 中译英 | 六级 |
| `buildEnToCnPrompt()` | 英译中 | 考研 |
| `buildIntensiveReadingPrompt(d)` | 双向 | 精读 |
| `buildCET6EssayTopicPrompt()` | — | 六级大作文 |
| `buildShortEssayTopicPrompt()` | — | 考研小作文 |
| `buildLongEssayTopicPrompt()` | — | 考研大作文 |
| `buildCET6EssayTemplatePrompt()` | — | 六级模板 |
| `buildShortEssayTemplatePrompt()` | — | 考研小作文模板 |
| `buildLongEssayTemplatePrompt()` | — | 考研大作文模板 |
| `buildTranslateGradePrompt(isEnToCn)` | 双向 | 翻译批改 |
| `buildEssayGradePrompt()` | — | 作文批改 |
| `buildVLImagePrompt()` | 识图 | 考研大作文 VL |

3 个分发函数：`buildTranslationPrompt()`、`buildEssayTopicPromptByMode()`、`buildEssayTemplatePromptByMode()`

### 6.6 消息气泡类型（18 种）

welcome / query / loading / result / error /
translate-result / en-to-cn-result / intensive-reading-result /
translate-analysis / long-essay-form /
essay-analysis / short-essay-analysis / long-essay-analysis /
essay-result / short-essay-result / long-essay-result /
translate-grade-result / essay-grade-result

### 6.7 错误处理

| 错误 | 提示 |
|---|---|
| 401 DeepSeek | API Key 无效，请前往设置更新 |
| 401 千问 | 千问 API Key 无效 |
| 429 | 请求过于频繁，请稍后重试 |
| 网络错误 | 网络连接失败，请检查网络 |
| 超时 30s (DeepSeek) | 请求超时 |
| 超时 25s (千问) | 识别超时 |
| JSON 解析失败 | AI 返回格式异常，请重试 |
| 空输入 | 输入框抖动 + 红字提示 |
| 非英文输入（词汇） | 仅支持英文内容 |
| 长文本（词汇 >15 词） | 引导切换精读模式 |
